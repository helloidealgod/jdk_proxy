10. 重要注意事项
地址空间：
基本地址空间：2K字（0-7777八进制）
通过BANK寄存器扩展：共36K字（可寻址）

数据格式：
15位数据 + 1位符号 + 1位奇偶 = 17位物理字
数值范围：-1 ≤ x < +1（分数表示）

特殊寻址模式：
INDEX指令修改下一条指令的地址字段
子程序返回：TC Q（使用Q寄存器）

溢出处理：
OVSK：溢出跳过下一条指令
TS指令：检测并存储溢出
