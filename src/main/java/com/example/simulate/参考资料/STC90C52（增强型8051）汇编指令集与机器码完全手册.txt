STC90C52（增强型8051）汇编指令集与机器码完全手册
STC90C52完全兼容标准8051指令集，并在此基础上扩展了新的指令。以下是详细分类的指令列表，包含机器码、汇编语法、执行周期和对标志位的影响：

机器码格式：xx表示8位立即数，dd表示直接地址，Ri表示R0或R1
周期数：基于12时钟模式（传统8051）
标志位：√影响，×不影响，?特定条件影响

六、STC扩展指令（部分型号支持）
STC90C52在传统8051基础上增加了一些实用扩展指令：
机器码 汇编指令    说明  功能
A5  NOP5    5周期空操作  延时用
85H SRC SFR MOV [Ri], [Rj]  寄存器间传送  快速数据搬移
支持  MOV Rn, @DPTR   DPTR间接送Rn   增强外部访问
支持  INC DPTR, n DPTR加n  快速指针移动
注意：扩展指令在不同STC型号中支持程度不同，请查阅具体数据手册。

七、标志位速查表
标志位 名称  位置  说明

AC  辅助进位    PSW.6   BCD码调整用
F0  用户标志0   PSW.5   用户自定义
RS1 寄存器区选择1 PSW.4   00=区0, 01=区1

P   奇偶标志    PSW.0   A中1的个数为奇则置1

八、寻址方式总结
立即寻址：MOV A, #30H
直接寻址：MOV A, 30H
寄存器寻址：MOV A, R0
寄存器间接：MOV A, @R0
变址寻址：MOVC A, @A+DPTR
相对寻址：SJMP rel
位寻址：SETB 20H.0
